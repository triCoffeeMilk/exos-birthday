
# EXOS HEROES BIRTHDAY

아래 기능은 구현중입니다...

1. 검색 시스템 (원하는 캐릭터 이름 검색 시, 생일을 알려줍니다.)
2. ~~캐릭터 이름 등급에 따라 색깔 변경~~
3. 캐릭터 이름에 마우스를 올릴 시, 해당 캐릭터의 정보 출력
    1. 이름, 생일, 속성, 유형, 등급, 얼굴, 나라 등...
